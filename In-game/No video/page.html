- if you cant see the .minecraft folder you need to enable the hidden files setting of your file explorer(top right for Cx)

- if the headset doesn't show up when you connect it to your pc, you disabled the mtp prompt, to re-enable it go to settings => system => developer => MTP prompt(or something similar)
